Visual Studio 安装的命令行参数示例

Visual Studio 安装的命令行参数示例

  • 安装c#桌面开发中文版
    vs_Community.exe –layout E:\vs2019 –add Microsoft.VisualStudio.Workload.ManagedDesktop –includeRecommended –lang en-US zh-CN

–add 后接参数称为workload

Visual Studio Community 组件目录

  • .NET 桌面开发
    ID： Microsoft.VisualStudio.Workload.ManagedDesktop

  • 使用 C++ 进行桌面开发
    ID： Microsoft.VisualStudio.Workload.NativeDesktop
    说明： 使用所选工具（包括 MSVC、Clang、CMake 或 MSBuild）生成适用于 Windows 的新式C++应用。

非关联组件
这些组件未包含在任何工作负荷中，但可以选择为单个组件。

Microsoft.VisualStudio.Component.VC.14.20.ATL
适用于 v142 生成工具的 v14.20 ATL C++ v14.20 ATL （x86 & x64）

Microsoft.VisualStudio.Component.VC.14.20.MFC
C++ v14.20 MFC for v142 生成工具（x86 & x64）

Microsoft.VisualStudio.Component.VC.14.20.x86.x64
MSVC v142 - VS 2019 C++ x64/x86 生成工具（v14.20）

常用
2019安装c#桌面开发
vs_Community.exe –layout E:\vs2019 –add Microsoft.VisualStudio.Workload.ManagedDesktop –includeRecommended –lang en-US zh-CN

图块快速插入

目的与功能

  1. 本插件开发目的是为了快速插入图块。

  2. 本插件加载后，可以直接在 AutoCAD 命令行通过输入图块名的方法，快速将指定图块插入到当前图形中。例如：图块文件夹中存放有文件 XG-1.dwg， 意义为鞋柜的第1种。当我们希望在当前图形中插入此图块时，只需在命令行中输入 XG-1 即可完成插入。

使用前准备

  1. 使用前应先本地电脑上建立图块文件夹。
  2. 缺省状态下，图块文件夹为桌面上为名为”图块“的文件夹。
  3. 在此文件夹下，依次按类别建立 图块类别文件夹，并存入图块文件。
  4. 图块类别文件夹以大写英文字母命名，如：XG代表鞋柜，CBG代表餐边柜，等等
  5. 图块文件按所属图块类别加短线加数字命名，如：XG-1.dwg、CBG-2.dwg等
  6. 图块文件夹如下：
桌面 │ 图块 │ ├──XG │   ├── XG-1.DWG │   ├── XG-2.DWG │   └── XG-3.DWG │ ├── CBG │   ├── CBG-1.DWG │   └── CBG-2.DWG │ └── DSG     ├── DSG-1.DWG     └── DSG-2.DWG

使用方法

  1. 本插件采用AutoLISP开发，可以适用各版本的AutoCAD（Windows版）。

  2. 插件文件为 TuKuai.VLX

  3. 使用前应先加载插件。加载方法详见：AutoCAD插件加载方法

  4. 完成加载后，可根据需要输入图块名，插入块。例如：

    • 输入“XG-1”，即可插入第1种鞋柜；
    • 输入“CBG-2”，即可插入第2种餐边柜。
